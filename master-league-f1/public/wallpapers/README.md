# Wallpapers do Dashboard

Esta pasta contém os wallpapers personalizados para o hero banner do dashboard de cada piloto.

## Estrutura de Arquivos

Cada equipe deve ter seu próprio wallpaper em estilo F1 (referências a corridas, estilo wallpaper):

- `f1-redbull.jpg` - Oracle Red Bull Racing
- `f1-ferrari.jpg` - Scuderia Ferrari
- `f1-mercedes.jpg` - Mercedes-AMG Petronas
- `f1-mclaren.jpg` - McLaren
- `f1-aston.jpg` - Aston Martin
- `f1-alpine.jpg` - Alpine
- `f1-haas.jpg` - Haas
- `f1-williams.jpg` - Williams
- `f1-sauber.jpg` - Stake F1 Team / Sauber
- `f1-vcarb.jpg` - Visa Cash App RB
- `f1-default.jpg` - Wallpaper padrão para pilotos sem equipe

## Especificações

- **Formato**: JPG ou PNG
- **Resolução recomendada**: 1920x1080 ou superior (16:9)
- **Estilo**: Wallpaper de corridas da F1, com referências visuais às equipes
- **Tamanho**: Otimizar para web (recomendado < 500KB por imagem)

## Exemplos de Temas

- Red Bull: Imagens de corridas com carros Red Bull, pódios, etc.
- Ferrari: Circuitos com carros vermelhos, momentos históricos
- Mercedes: Corridas com carros prateados/pretos, vitórias
- McLaren: Laranja e azul, momentos de glória
- E assim por diante...

## Nota

As imagens devem ser adicionadas manualmente nesta pasta. O sistema automaticamente detectará e usará o wallpaper correto baseado na equipe mais recente do piloto.
